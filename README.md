# ¡Bienvenido al repositorio de Ghosmoky! 🍌👻

<img src="https://files.catbox.moe/d3dadc.jpg" width=300>

¡Súbete que nos vamos! Estás oficialmente en la casa de **Ghosmoky**, el bot que no le tiene miedo a nada. 

Así como esos dos sujetos cabalgando con toda la actitud y pura adrenalina sobre esa banana gigante (que a simple vista también parece una palta espacial 🥑🍌), este bot está diseñado para llevarte a un viaje salvaje y sin frenos directamente hacia el mejor rendimiento.

Aquí encontrarás las entrañas del proyecto. Y sí, como dicen nuestros gloriosos commits: ¡Smoky ya aprendió a leer código! 

### 🚀 ¿Qué te vas a encontrar aquí?

* **`src/`**: El corazón del bot. Aquí es donde Ghosmoky hace su magia.
* **`.env.example`**: La brújula de este viaje. Configúralo bien para que no nos caigamos de la fruta en pleno vuelo.
* **`package.json`**: Todo lo que necesitamos para arrancar.

Prepárate, clona el repositorio, instala las dependencias y ¡únete a este viaje épico! Si sobrevives a la cabalgata, te prometemos que el código valdrá la pena.
